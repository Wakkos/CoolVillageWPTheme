Buttons

We have the component block into a sass extend and we extend it on the modifier.

%btn {
   display: inline-block;
   line-height: $baselineheight * 2;
   color: $buttoncolor;
   text-align: center;
   vertical-align: middle;
   font-size: fs(normal);
   white-space: nowrap;
   background-color: $buttonbackground;
   border-top: $buttonborderwidth $buttonborderstyle $buttonbordercolor;
   border-right: $buttonborderwidth $buttonborderstyle $buttonbordercolor;
   border-bottom: ($buttonborderwidth * 2) $buttonborderstyle |$buttonbordercolor;
   border-left: $buttonborderwidth $buttonborderstyle $buttonbordercolor;
   cursor: pointer;
   margin:0;
   padding-top:   0;
   padding-bottom:0;
   padding-right:1em;
   padding-left: 1em;
   border-radius: $radius;
   @include transition(all $hovertransition ease-in);
   /**
   * Nadie quiere un bot√≥n con texto subrayado cuando
   * hacemos hover.
   */
   &:hover {
       text-decoration: none;
       color: $buttoncolor;
   }
   &:active,
   &:focus{
       outline:none;
   }
   &:visited {
   color: $buttoncolor;
   }
}

All buttons comes with the extend %btn block to look like a button, then, we add the modifier: .btn--modifier. You should extend the extender in every partial that uses the buttom.

All these %extend are made to simplify CSS as detailed in Una Kravets method Atomic OOBEMITSCSS.

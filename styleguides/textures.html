Alert Box

We have the component block into a sass extend and we extend it in the modifier.

%alert {
   margin-bottom : $baselineheight;
   padding : 8px 35px 8px 14px;
   background-color : $color-alert;
   border : $alertborderwidth $alertborderstyle $alertbordercolor;
   font-family : $basefont;
   font-size: fs(small);
   color : $alerttext;
   border-radius: $radius;
   position: relative;
   .alert--heading {
       color : inherit;
   }
   .close {
       position : absolute;
       top : -2px;
       right : 2px;
       float : right;
       font-size: fs(small);
       font-weight : bold;
       line-height : 18px;
       color : inherit;
   }
}
